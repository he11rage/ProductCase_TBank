# Анализ данных сервиса аренды самокатов Т-Банк (сезон 2024)

**Описание**:
- Проект посвящен разведочному анализу данных (EDA) по поездкам на самокатах сервиса Т-Банк за сезон 2024 (апрель-октябрь).
- Цель - выявить ключевые паттерны поведения пользователей, построить продуктовые гипотезы и оценить долю рынка сервиса.
